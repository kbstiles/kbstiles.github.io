Main Premise

The task was to create a fully functioning lodge booking application. It needed to allow customers to view and book lodges as well as view and finalize receipts. As for employees, they needed to be able to view, edit, add and remove lodges. We needed to handle data manipulation through an SQL Database and run segments through threading to make the program run smoothly.
